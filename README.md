🎙️ AI-Voice-Assistant

A smart Python-based voice assistant that listens, understands, and responds — just like a mini AI companion on your desktop.

✨ Features

🔊 Speech Recognition

Listens to your voice commands

Converts speech to text using Google Speech API

🗣️ Text-to-Speech

Responds naturally using a speech synthesis engine

🎵 YouTube Integration

Plays songs directly on YouTube using pywhatkit

⏰ Time Inquiry

Instantly tells the current time

📚 Wikipedia Search

Fetches short summaries from Wikipedia


🖥️ GUI Interface

Interactive and user-friendly interface built with Tkinter

🛠️ Technologies Used
Technology	Purpose
🐍 Python	Core programming language
🖼️ Tkinter	GUI interface
🎤 SpeechRecognition	Voice input processing
🌐 Google Speech API	Speech-to-text engine
🔈 pyttsx3	Text-to-Speech
▶️ pywhatkit	YouTube search & playback
📖 Wikipedia API	Information retrieval
😄 pyjokes	Random joke generator
